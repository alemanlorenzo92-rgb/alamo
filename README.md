# ÁLAMO - Catálogo de Indumentaria

Este es el catálogo digital de indumentaria para el emprendimiento **ÁLAMO**.

## Características
- Diseño minimalista y elegante.
- Catálogo interactivo con integración direta a WhatsApp.
- Totalmente responsivo para móviles y escritorio.
- Fondo dinámico con micro-animaciones en CSS.

## Tecnologías Utilizadas
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- [serve](https://www.npmjs.com/package/serve) (para hosting estático)

## Despliegue en Railway
Este repositorio está configurado para ser desplegado automáticamente en **Railway**.

1. Sube este código a GitHub.
2. Conecta tu cuenta de Railway con GitHub.
3. Crea un "New Project" -> "Deploy from GitHub repo".
4. Selecciona este repositorio.
5. Railway detectará el `package.json` e instalará `serve` para mostrar la página.
